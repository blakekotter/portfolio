Projects

Banking App

The Banking App is built to simulate banking tasks for admins, employees, and customers. It provides specific dashboards for each user role, making it easy to manage accounts and complete transactions.

  • Admin Dashboard: Oversee the banking system, manage users, and transactions.
  • Employee Dashboard: Manage customer accounts and transactions.
  • Customer Dashboard: Access account details, transfer funds, and view transaction history.

Mongo Accounts

The Mongo Accounts app lets users sign up, log in, and manage their account information. It’s built with MongoDB and Express on the backend, while React powers the frontend for an interactive user experience.

Hangman Game

This project is a word-guessing game implemented using the MERN stack. The game features a user interface, random word generation, and session management to track high scores.

  • Random Word Generation: Words are chosen randomly from a database, making each game unique.
  • Session Management: Tracks player scores and game state across sessions.

Tic-Tac-Toe

This project presents a digital version of the TicTacToe game, built using modern front-end frameworks. It includes a simple interface, responsive design, and logic to track and display the game's progress.

Four in a Row

This project recreates the classic Four in a Row (Connect Four) game, with an emphasis on algorithmic game logic and user experience. The game features a dynamic interface, responsive design, and easy-to-use controls.
