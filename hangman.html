Project Overview

The Hangman Game is a classic word-guessing game developed using the MERN stack (MongoDB, Express.js, React.js, Node.js). The game challenges players to guess a hidden word by suggesting letters one at a time.

Key Features

  • Random Word Generation: Each game starts with a randomly selected word from a pre-defined database, ensuring a fresh challenge with every playthrough.
  • Session Management: The game tracks player progress and scores across sessions, allowing users to return and continue playing without losing their progress.
